## Instructions

1. Start the application: `mvn spring-boot:run`
2. Log in as **admin / admin123**
3. Navigate to each module and capture screenshots
4. Save screenshots in this directory with the filenames listed above
5. Ensure screenshots show real database data (seeded automatically on first run)

## Viva Tips

- Use screenshots that demonstrate working CRUD operations
- Capture both Admin and Employee views where applicable
- Include screenshots with meaningful data (not empty tables)
